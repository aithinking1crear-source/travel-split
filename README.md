# 🧳 Travel Split

App para dividir gastos de viaje entre varias personas, considerando los días que estuvo cada una.

## Funcionalidades

- 📅 Define cuántos días estuvo cada participante
- 💰 Registra gastos: descripción, monto y quién pagó
- 🧮 Calcula el valor por persona por día automáticamente
- 💸 Muestra las transferencias mínimas para saldar las cuentas
- ✏️ Edita gastos y participantes en cualquier momento
- 📚 Guarda historial de viajes en el navegador

## Tecnología

- React + Vite
- Almacenamiento local del navegador (localStorage)
